Course Description

This course provides an introduction to the field of Sequential Decision Making. We will cover the following topics:

  1. The Multi Armed Bandit Model and algorithms such as Upper Confidence Bound (UCB), Thompson Sampling and EXP3.
  2. The Contextual Bandit model and algorithms such as EXP4 and algorithms for linear models such as OFUL.
  3. Online Model Selection algorithms and the principle of regret balancing.
  4. Constrained Bandit Problems. Doubly Optimistic and Pessimistic-Optimistic methods.
  5. Statistical dimensions for bandits and contextual bandits with nonlinear function approximation. Eluder dimension, dissimilarity dimension and the Decision Estimation Coefficient (DEC).

The assessment tools consist of five parts:

  1. Homeworks including both written problems and coding problems in Python, using Jupyter notebooks.
  2. In-class participation
  3. A midterm.
  4. A final exam.
  5. A final project. More details about this below.

Learning Objectives

  • Understand the fundamentals of reinforcement learning.
  • Implement and evaluate basic bandit algorithms.
  • Understand the practical use of bandit algorithms for personalized recommender systems.
  • Learn basic metrics for evaluation.
